# Shared Utilities & Middleware

This package contains shared code used across all microservices.

## Contents

### Middleware
- `auth.js` - Authentication and authorization middleware

### Config
- `cloudinary.js` - Cloudinary configuration and upload helpers

### Utilities
- `imageUpload.js` - Image upload/delete functions
- `responseFormatter.js` - Standard API response formats
- `errorHandler.js` - Error handling utilities
- `validation.js` - Input validation functions
- `logger.js` - Logging utility
